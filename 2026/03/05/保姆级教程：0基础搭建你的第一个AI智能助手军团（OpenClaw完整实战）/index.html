保姆级教程：0基础搭建你的第一个AI智能助手军团（OpenClaw完整实战）

想要一个24小时在线的AI助手团队？这篇文章教你从零搭建

你是否遇到过这些困扰：

  • 信息太多处理不过来，需要花大量时间整理和调研
  • 重复性工作太多，写代码、写文章、发布内容都要亲力亲为
  • 想要多个AI助手协同工作，但不知道怎么配置

如果有一支AI助手军团，能够24小时帮你调研信息、编写代码、创作内容，该多好？

今天分享一个完整的 OpenClaw AI Agent 搭建教程，从环境安装到组建多Agent团队，预计1-2小时即可完成。即使你是零基础小白，跟着步骤操作也能成功。

OpenClaw是什么？能帮你做什么？

OpenClaw 是一个开源的 AI Agent 框架，可以让你创建多个具有不同能力的智能助手：

  • ✅ 记忆能力：记住你的对话历史和偏好
  • ✅ 编程能力：自动编写和调试代码
  • ✅ 搜索能力：自动调研信息、浏览网页
  • ✅ 协同工作：多个Agent可以组成团队分工协作

适合人群：

  • 程序员：需要AI助手帮助编码和调研
  • 内容创作者：需要AI帮助写作和发布
  • 知识工作者：需要AI帮助整理信息和自动化任务

二、创建你的第一个AI助手（15分钟）

1. 为什么选择 Telegram

我们使用 Telegram 作为与AI助手的通讯渠道，原因是：

  • 配置简单，对新手友好
  • 支持群聊，方便多个Agent协同
  • 跨平台，手机和电脑都能用

2. 创建 Telegram Bot

打开 Telegram，搜索 @BotFather：

  1. 发送 /start 开始对话
  2. 发送 /newbot 创建新机器人
  3. 输入 Bot 显示名称（例如：AI助手大总管）
  4. 输入 Bot 用户名（必须以 _bot 结尾，例如：my_ai_assistant_bot）

成功后，BotFather 会返回一个 Token（格式类似：7691627338:AAHo9ix-evUZaz2FgmVAF9juHohsSSX3KOa），请保存好这个Token。

同时注意消息中的 t.me/xxx 链接，点击即可进入与你的机器人的对话页面。

九、组建AI助手军团（20分钟）

1. 什么时候需要多个Agent？

应该创建多个Agent的情况：

✅ 上下文会持续积累且互不干扰

  • 例子：调研Agent积累行业知识，写作Agent积累写作风格

✅ 需要不同的”记忆”和”专业知识”

  • 例子：技术调研Agent vs 市场调研Agent

✅ 工作流程完全独立

  • 例子：内容创作流程 vs 代码开发流程

不应该拆分的情况：

❌ 只是工具不同，但上下文相同

  • 错误：前端Agent、后端Agent（都是编程，统一用Codex）

❌ 只是输出格式不同

  • 错误：小红书Agent、公众号Agent（都是写作，只是Skills不同）

❌ 任务之间需要频繁共享信息

  • 错误：需求分析Agent、架构设计Agent（应该是一个Agent的不同阶段）

2. 创建多Agent示例

以创建两个Agent为例：

  • 佩奇：负责调研
  • 格雷厄姆：负责写作

步骤1：创建Bot Token

参照前面的方式，为两个Agent分别创建Bot Token。

步骤2：设置群聊权限

对BotFather输入：

  1. /setprivacy
  2. 选择你的Bot
  3. 选择 Disable

这样Bot就能读取群内消息了。

步骤3：创建群组

  1. 点击 New Group 创建群组
  2. 输入群组名称（例如：AI工作室）
  3. 将你的Bot拉入群组

步骤4：获取ID

搜索 @userinfobot 机器人：

  1. 选择 user，点击自己的头像（获取用户ID）
  2. 选择 Group，点击你创建的群组（获取群组ID）
  3. 保存这两个ID

步骤5：配置Agent

使用这个配置模板：
https://github.com/bozhouDev/openclaw_agent_create_prompt/blob/main/Agent-create-prompt.md

将模板内容修改后，让你的主AI助手来进行配置。
