OpenClaw 是什么？为什么要接入飞书？

OpenClaw 简介

OpenClaw 是一款开源的个人 AI Agent 系统，可以运行在你的个人电脑或服务器上。它最大的特点是：

  • 完全属于你的私人助理：拥有长期记忆、能力可持续进化
  • 权限更高：能直接操作你的电脑，本地运行
  • 7×24 小时主动干活：不是被动回答问题，而是主动完成任务

把 OpenClaw 比作”可扩展工具箱”再合适不过：能接模型、能接工具、能接各种服务，就像给自己雇了个不抱怨、技能极多的数字同事。

完整安装教程（跟着做就能成功）

前提条件

  1. 已安装 OpenClaw

    • Linux/MacOS：需要 openclaw 2026.2.26 及以上
    • Windows：需要 openclaw 2026.3.2 及以上
    • 查看版本：openclaw -v
    • 升级命令：npm install -g openclaw

  2. 创建飞书机器人应用

步骤 1：创建飞书应用

  1. 登录 飞书开放平台
  2. 点击”创建企业自建应用”
  3. 配置应用名称、描述和图标，点击”创建”

步骤 2：添加机器人能力

  1. 在左侧菜单选择 “应用能力 > 添加应用能力”
  2. 选择”按能力添加”页签
  3. 点击”机器人”能力卡片的”添加”按钮

步骤 3：配置权限

  1. 在左侧菜单选择 “开发配置 > 权限管理”
  2. 点击”批量导入/导出权限”
  3. 在”导入”页签中，粘贴权限配置(详见飞书官方文档)
  4. 点击”确认新增权限”并”申请开通”

步骤 4：发布应用

  1. 点击顶部”创建版本”
  2. 配置版本号和更新说明
  3. 点击”保存”并”确认发布”

步骤 5：获取配置信息

  1. 在左侧菜单选择 “基础信息 > 凭证与基础信息”
  2. 记录 App ID 和 App Secret（后面安装插件时需要）
