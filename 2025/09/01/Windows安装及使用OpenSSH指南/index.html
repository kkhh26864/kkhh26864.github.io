7. 常见问题排查

无法连接：

  • 检查 sshd 服务是否正在运行：Get-Service sshd。
  • 确保防火墙允许 22 端口（或自定义端口）。
  • 检查 sshd_config 文件是否有语法错误。

权限问题：

  • 确保 C:\ProgramData\ssh\sshd_config 和 .ssh\authorized_keys 文件具有正确的权限。通常只有 SYSTEM 和 Administrators 应具有完全控制权限。
  • 可以使用 icacls 命令设置权限：

日志查看：
OpenSSH 的日志通常位于 Windows 事件查看器中，或者检查 C:\ProgramData\ssh\logs 目录。
